# @fuse_ui/avatar

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/avatar
```

## Usage

```ts
import { FuseAvatarComponent } from '@fuse_ui/avatar';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `rxjs` >= 7.4.0
