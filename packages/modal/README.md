# @fuse_ui/modal

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/modal
```

## Usage

```ts
import { FuseModalComponent, FuseModalService, FuseConfirmDialogComponent } from '@fuse_ui/modal';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `@angular/cdk` >= 18.0.0
- `rxjs` >= 7.4.0
