# @fuse_ui/checkbox

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/checkbox
```

## Usage

```ts
import { FuseCheckboxComponent, FuseCheckboxGroupComponent, FuseRadioComponent, FuseRadioGroupComponent } from '@fuse_ui/checkbox';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `@angular/forms` >= 18.0.0
- `rxjs` >= 7.4.0
