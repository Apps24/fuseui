# @fuse_ui/pagination

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/pagination
```

## Usage

```ts
import { FusePaginationComponent } from '@fuse_ui/pagination';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `rxjs` >= 7.4.0
- `@fuse_ui/icon` >= 0.1.0
