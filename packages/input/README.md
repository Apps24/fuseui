# @fuse_ui/input

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/input
```

## Usage

```ts
import { FuseInputComponent, FuseTextareaComponent } from '@fuse_ui/input';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `@angular/forms` >= 18.0.0
- `rxjs` >= 7.4.0
