# @fuse_ui/form-field

> Part of [Fuse UI](https://github.com/Apps24/fuseui) — Angular 18+ / Ionic 8+ component library

## Installation

```bash
npm install @fuse_ui/form-field
```

## Usage

```ts
import { FuseFormFieldComponent } from '@fuse_ui/form-field';
```

## Peer Dependencies

- `@angular/core` >= 18.0.0
- `@angular/common` >= 18.0.0
- `rxjs` >= 7.4.0
- `@fuse_ui/label` >= 0.1.0
